## How to Run

```bash
# 1. Start PostgreSQL
docker compose up -d postgres

# 2. Start Spring Boot backend  (tables auto-created by Hibernate on first run)
cd backend
mvn spring-boot:run
# Server on http://localhost:3000

# 3. Start React frontend
cd frontend
npm run dev
# App on http://localhost:5173
```
